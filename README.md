# Apple Pay API (apple-pay)
Apple Pay enables secure, simple checkouts in your app or website. Customers can purchase physical goods and services using the payment cards they've securely stored in their Wallet app.

**URL:** [Visit APIs.json URL](https://developer.apple.com/apple-pay/)

## APIs

### Apple Pay JS API
Use Apple Pay on the web to accept payments in Safari on iOS, iPadOS, and macOS. It allows users to authorize payments using Touch ID or Face ID.
